# Animations Folder

This folder contains Lottie animations for the app.

## Required Animations

1. meditation.json - For breathing exercise
2. timer.json - For math challenge timer
3. book.json - For word builder game
4. brain.json - For memory cards game
5. star.json - For motivational quotes
6. music.json - For music player

## Where to get animations

Download free animations from:
- https://lottiefiles.com/
- https://lordicon.com/

Search for:
- "meditation breathing"
- "timer countdown"
- "book reading"
- "brain memory"
- "star achievement"
- "music equalizer"

## Usage

Animations are used in FunZoneScreen with:
```dart
Lottie.asset('assets/animations/meditation.json')
```
